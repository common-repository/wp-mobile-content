A simple plugin which lets you add Mobile Only content on website/blog

== Description ==
Display content for only users whos are on a mobile device. Use [mobile][/mobile] for this.

== Installation ==
1. Upload the file " to the \"/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Plugins\" menu in WordPress.


== Frequently Asked Questions ==
= Shortcode is not working what do i do? =
Try and use the text mode instead of visual mode and it should work. If any issues please open a support here in wordpress.org
